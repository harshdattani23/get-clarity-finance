## 🎯 Project Overview

Get Clarity Finance is a comprehensive AI-powered platform built for the **[Securities Market Hackathon at GFF 2025](https://uat.globalfintechfest.com/gff-hackathons/sebi-hackathon)**, powered by **BSE, CDSL, KFINTECH, NSDL and SEBI**. Our solution directly addresses the critical fraud prevention challenges outlined in the hackathon while enhancing investor education and market accessibility.

## 🎯 **SEBI Hackathon Problem Statement: Fraud Prevention**

### 🚨 **The Crisis We're Solving**

The hackathon specifically identifies these critical fraud tactics that our platform combats:

#### **1. Social Media & Digital Fraud**
- **🎭 Deepfake Content**: Circulation of deepfake videos/audios of corporate leaders
- **📋 Fabricated Documents**: Fake regulatory letters of approvals/fines
- **📱 WhatsApp/Telegram Scams**: Misleading stock tips and pump-and-dump schemes
- **🔄 Coordinated Manipulation**: Orchestrated buying/selling to manipulate prices

#### **2. Impersonation & False Identity**
- **🏢 Fake Intermediaries**: Fraudsters posing as registered merchant bankers/custodians
- **💰 IPO Scams**: False promises of firm allotments through social media
- **📱 Fake Trading Apps**: Apps mimicking trusted firms with fictitious trades
- **💸 Portfolio Manipulation**: Showing fake gains to encourage larger investments

#### **3. Corporate Announcement Fraud**
- **📢 False Announcements**: Listed companies issuing misleading corporate announcements
- **📰 Market Manipulation**: False narratives leading to retail investor losses
- **🔍 Information Asymmetry**: Exploiting lack of verification mechanisms

### 🛡️ **Our Direct Solution Response**

**As specified in the hackathon requirements, we deliver:**

✅ **Multi-Platform Scanning**: App that scans online platforms and identifies suspicious investment offers, videos, audios, and documents

✅ **Regulatory Verification**: Real-time verification of advisor credentials against SEBI regulatory databases

✅ **Risk Assessment Engine**: Advanced prototype that flags high-risk promotions and advisors

✅ **User-Friendly Interface**: Intuitive platform for investors to check legitimacy instantly

✅ **Social Media Monitoring**: Solution monitoring WhatsApp, Telegram, and social platforms for suspicious stock tips

✅ **Market Activity Correlation**: Links suspicious tips to unusual market activity patterns

✅ **Regulatory Dashboard**: Provides actionable insights for regulators and exchanges

✅ **Privacy Compliance**: Ensures user privacy while maintaining regulatory standards

✅ **Document Verification**: OCR + AI analysis to verify corporate announcements authenticity

1. **Clone & Install**
   ```bash
   git clone https://github.com/your-repo/get-clarity-finance.git
   cd get-clarity-finance
   npm install
   ```

2. **Environment Setup**
   ```bash
   cp .env.local.example .env.local
   # Edit .env.local with your API keys
   ```

3. **Database Setup**
   ```bash

   npx prisma generate

   npx prisma migrate dev

   npm run seed:achievements
   ```

   ```bash
   npm run dev
   # Open http://localhost:3000
   ```

## 🚀 **Deployment**

### 🐳 **Docker Deployment**

```dockerfile
# Multi-stage build for optimal image size
FROM node:20-alpine AS deps
WORKDIR /app
COPY package*.json ./
RUN npm ci

FROM node:20-alpine AS builder
WORKDIR /app
COPY --from=deps /app/node_modules ./node_modules
COPY . .
RUN npx prisma generate
RUN npm run build

FROM node:20-alpine AS runner
WORKDIR /app
COPY --from=builder /app/.next/standalone ./
EXPOSE 3000
CMD ["node", "server.js"]
```

### ☁️ **Google Cloud Run**

